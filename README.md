## Getting Started

### Prerequisites

Run this on your terminal.
* npm
  ```sh
  npm init
  ```
  
* express
  ```sh
  npm install express
  ```
  
* nodemon
  ```sh
  npm install --save-dev nodemon
  ```
  
* mongoose
  ```sh
  npm install mongoose
  ```
  
* dotenv
  ```sh
  npm install dotenv
  ```
  
* hapi/joi
  ```sh
  npm install @hapi/joi
  ```
  
* bcryptjs
  ```sh
  npm install bcryptjs
  ```
  
* jsonwebtoken
  ```sh
  npm install jsonwebtoken
  ```
